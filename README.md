# xchat-plugins
Plugins y scripts para el cliente de IRC XChat

- ./asciihuayra :
  - Ascii Art de vacas y panaderos
- ./rsshuayra :
  - Comunica en el canal IRC #huayragnulinux de Freenode la publicación de nuevos posts en los foros de Huayra GNU/Linux (oficial y comunidad de usuarios). Adaptable.
